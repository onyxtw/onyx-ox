## 🌌 Overview

**ONYX‑OX** 是 ONYX 宇宙的核心節點（Node‑0），  
負責執行：

- ONYX‑OS（語義操作系統）
- ONYX‑AI（語義推理引擎）
- ONYX‑Engine（行為生成引擎）
- ONYX‑Self（自我模型）
- ONYX‑Network（節點間語義交換）
- ONYX‑Hub（API / Gateway）
- ONYX‑UI（介面層）

本倉庫是 **Hybrid Universe Architecture（HΩ）** 的基礎實作，  
也是 ONYX‑OX 文明系統的第一個公開節點。

## 🧠 Core Concepts

### **Semantic Substrate（語義基底）**
所有語義事件（events）都在此流動。

### **Semantic Operators（語義算子）**
所有模組的共同基底。

### **Civilization Layer（文明層級）**
節點的語義文明狀態（C‑0 → C‑1 → C‑2…）。

### **Five Semantic Modules**
- Action（行動）
- Energy（能量）
- Structure（結構）
- Transform（轉化）
- Emergence（湧現）
